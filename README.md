# K-Fold Image Classification

## Overview
This project implements image classification using K-Fold Cross Validation in PyTorch.

## Features
- K-Fold training (5 folds)
- Custom dataset loader
- CNN model
- Evaluation using average accuracy

## Results
Final Accuracy: ~77%

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run training:
   python src/train.py
